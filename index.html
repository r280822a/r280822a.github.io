Python Code Challenges

The code below works, but it takes up too many lines or otherwise needs improving. Can you optimize it?

Challenge 1

Goal: Reduce this 5-line program to just 2 lines.


print("Hello World")
print("Hello World")
print("Hello World")
print("Hello World")
print("Hello World")

Your attempt:

Solution: Use a FOR loop instead of repeating the code!


for i in range(5):
    print("Hello World")

Challenge 2

Goal: You don't need to reduce the number of lines in this code but do you notice the mistake?


coins = 60
if coins >= 50:
    return True
else if coins < 50:
    return False

Your attempt:

Solution: If `coins >= 50` is false then coins must be less than 50. So no need for an else if, just use a regular else statement:


coins = 60
if coins >= 50:
    return True
else:
    return False

Challenge 3

Goal: Reduce this 5-line program to just 2 lines.


numbers = [5, 10, 15, 20]
total = 0
for num in numbers:
    total = total + num
print(total)

Your attempt:

Solution: Python has built-in functions that do the heavy lifting for you. Use `sum()` here.


numbers = [5, 10, 15, 20]
print(sum(numbers))
